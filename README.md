## SQL-Project
Airline Database SQL Project
📌 Project Overview

* This project is based on an Airline Reservation Database designed to manage flight bookings, passengers, tickets, aircraft details, airports, boarding passes, and flight schedules.

* The database simulates a real-world airline booking system where one booking can contain multiple passengers, each receiving separate tickets with connected or round-trip flight segments.

* This project demonstrates strong skills in SQL database design, querying, joins, aggregation, and data analysis.

### Booking Analysis
* Total bookings made
* Revenue by month
* Highest booking routes
### Flight Analysis
* Delayed flights count
* On-time performance
* Most frequent routes
###  Seat Occupancy
* Occupancy percentage by flight
* Class-wise seat usage
###  Passenger Analysis
Frequent flyers
Top spending customers
###  Airport Analysis
* Flights per airport
* Most connected cities
### Technologies Used
* SQL
* Relational Database Design
* ER Modeling
